# Promise


## Description

This Reposiory is about PromiseNFT Blockchain/NFT Backend Application Using Klaytn Blockchain Network.  
  - Use KIP(Klaytn Compatible Token)-17 for NFT minting

## 핵심 기능

- 카이카스 지갑 로그인
- 약속
    - 만들기, 삭제하기, 수정하기
    - 공유하기
    - 참여하기
    - 기록하기

## Getting Start

1. 개발 서버 시작

```sh
yarn install
yarn start
```
